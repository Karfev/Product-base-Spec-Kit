# domains/is-ontology/

L1-домен: **Онтология архитектуры информационных систем**

Домен кодирует понятия, структуры и принципы из «Онтологии и методологических рекомендаций по использованию при проектировании архитектуры АИС» (ГНИВЦ / ФНС, 2025). Основан на ArchiMate 3.2 и ГОСТ Р 57100-2016.

**Использование:** Ссылайтесь на артефакты этого домена из `design.md` инициатив, работающих с системами АИС-класса. При создании NFR-требований (type: nfr) используйте `nfr/domain-nfr.md` для выбора `iso_characteristic`.

## Ключевые концепции

**Три архитектурных слоя:**
- **Слой деятельности** (жёлтый) — бизнес-процессы, участники, события
- **Слой приложений** (бирюзовый) — компоненты ПО, данные, интерфейсы
- **Технологический слой** (зелёный) — узлы, системное ПО, артефакты

**Классификационные коды:**
- Системы: `С.М.М` (малый) / `С.М.С` (средний) / `С.М.Б` (большой)
- Подсистемы: `ПС.Т.И` (инфраструктурная) / `ПС.Т.ПТ` (программно-технологическая) / `ПС.Т.П` (прикладная)
