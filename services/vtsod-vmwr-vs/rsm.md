## Общая схема РСМ

```mermaid
graph TD
    SVC["Услуга ВЦОД-VMWR-VS\n(ВЦОД Потребителя)"]

    subgraph "Уровень платформы"
        VMW_COMP["Платформа виртуализации VMware\n(кластер вычислений)"]
        VMW_MGMT["Платформа виртуализации VMware\n(кластер управления)"]
        PORTAL["Единый портал + терминальные серверы\n(консоль управления)"]
    end

    subgraph "Уровень инфраструктуры"
        COMPUTE["Вычислительная инфраструктура\n(серверы)"]
        STORAGE["Инфраструктура хранения данных\n(СХД, SAN)"]
        NETWORK["Телекоммуникационная инфраструктура\n(коммутаторы, маршрутизаторы, МСЭ)"]
        SEC["Подсистема ИБ\n(VMware Security)"]
        MON["Подсистема мониторинга"]
        INFRA_SVC["Подсистема инфраструктурных сервисов\n(DNS, NTP, AD)"]
        BACKUP["Подсистема резервного копирования"]
    end

    subgraph "Внешние поставщики"
        DC["Инженерная инфраструктура ЦОД\n(ООО РТК-ЦОД)"]
        ISP["Внешние каналы связи\n(ПАО Ростелеком)"]
    end

    SVC --> VMW_COMP
    SVC --> PORTAL
    VMW_COMP --> COMPUTE
    VMW_COMP --> STORAGE
    VMW_COMP --> NETWORK
    VMW_MGMT --> VMW_COMP
    PORTAL --> VMW_MGMT
    PORTAL --> NETWORK
    SEC --> VMW_COMP
    SEC --> NETWORK
    MON --> VMW_COMP
    MON --> STORAGE
    MON --> NETWORK
    INFRA_SVC --> VMW_MGMT
    BACKUP --> STORAGE
    COMPUTE --> DC
    STORAGE --> DC
    NETWORK --> DC
    NETWORK --> ISP
```

## РСМ — Телекоммуникационная инфраструктура

Обеспечивает сетевую связность внутри ВЦОД, межсетевое экранирование и доступ в Интернет.

```mermaid
graph LR
    subgraph "Телекоммуникационная инфраструктура"
        SW_AGG["Агрегационные коммутаторы"]
        SW_TOR["ToR-коммутаторы (Top-of-Rack)"]
        FW["Виртуальные МСЭ\n(Stateful-Firewall)"]
        GW["Шлюзы / маршрутизаторы"]
        VLAN_POOL["Пул VLAN\n(изолированные сегменты)"]
    end
    SW_TOR --> SW_AGG
    SW_AGG --> GW
    FW --> SW_AGG
    VLAN_POOL --> SW_TOR
    GW --> ISP["Внешние каналы связи\n(Ростелеком)"]
```

| Компонент | Тип | Ответственный |
|---|---|---|
| Агрегационные коммутаторы | Физическое оборудование | ООО «РТК-ЦТ» |
| ToR-коммутаторы | Физическое оборудование | ООО «РТК-ЦТ» |
| Виртуальные МСЭ | Виртуальный компонент | ООО «РТК-ЦТ» |
| Шлюзы/маршрутизаторы | Физическое оборудование | ООО «РТК-ЦТ» |
| Внешние каналы связи | Сервис | ПАО «Ростелеком» |

## РСМ — Инфраструктура хранения данных

Обеспечивает три типа дискового хранилища с разными характеристиками IOPS и задержки.

```mermaid
graph LR
    subgraph "Инфраструктура хранения данных"
        SAN["Сеть хранения данных (SAN)"]
        SSD_NVMe["СХД тип 3\n(NVMe, latency ≤5мс, 2000 IOPS/ТБ)"]
        SSD["СХД тип 2\n(SSD, latency ≤20мс, 200 IOPS/ТБ)"]
        HDD["СХД тип 1\n(HDD, latency ≤100мс, 20 IOPS/ТБ)"]
    end
    SAN --> SSD_NVMe
    SAN --> SSD
    SAN --> HDD
```

## РСМ — Платформа виртуализации VMware (кластер вычислений)

```mermaid
graph LR
    subgraph "Кластер вычислений"
        vCenter["vCenter Server"]
        ESXi_1["ESXi Гипервизор 1"]
        ESXi_N["ESXi Гипервизор N"]
        VTSOD["ВЦОД Потребителя\n(VM Pool, Resource Pool)"]
    end
    vCenter --> ESXi_1
    vCenter --> ESXi_N
    ESXi_1 --> VTSOD
    ESXi_N --> VTSOD
```

**Параметры лимитов платформы VMware:**

| Параметр | Мин | Макс |
|---|---|---|
| vCPU для одной ВМ | 1 | 48 |
| vRAM для одной ВМ | 1 Гб | 768 Гб |
| vHDD для одной ВМ | 1 Гб | 15 ТБ |
| Кол-во vHDD для одной ВМ | 1 | 26 |
| vNIC для одной ВМ | 1 | 10 |
| Пропускная способность vNIC | 1 Гбит/с | 10 Гбит/с |
| VLAN | 10 | 400 |
| Пропускная способность МСЭ | 2 Гбит/с | 10 Гбит/с |
| Терминальных серверов | 1 | 6 |
| Сетевых шлюзов | 1 на VLAN | — |
