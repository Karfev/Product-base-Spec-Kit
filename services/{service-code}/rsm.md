## Общая схема РСМ

```mermaid
graph TD
    subgraph "Услуга: {service-code}"
        SVC["{Краткое наименование}\nуслуги"]
    end

    subgraph "Уровень {N}: {Наименование уровня}"
        L1_1["{Подсистема / компонент 1}"]
        L1_2["{Подсистема / компонент 2}"]
    end

    subgraph "Уровень {N-1}: {Наименование уровня}"
        L2_1["{Ресурс 1}"]
        L2_2["{Ресурс 2}"]
    end

    SVC --> L1_1
    SVC --> L1_2
    L1_1 --> L2_1
    L1_2 --> L2_2
```
